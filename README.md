# QN2-VUPay

A Java application developed for Victoria University Kampala (VU) that allows applicants to enter a Module Code and get the corresponding course details returned to the screen.

## Project Structure

- `VUCourses.java` - Class that defines the course attributes (Module Name, Module Code, Tuition)
- `VUPay.java` - Main class that runs the application and handles user input

## Modules Available

| Module                        | Code | Tuition (UGX) |
|-------------------------------|------|---------------|
| BSc. Software Engineering     | BSF  | 900,000       |
| BSc. Information Technology   | BIT  | 750,000       |
| BSc. Computer Science         | BCS  | 800,000       |
| BSc. Computer Engineering     | BCE  | 950,000       |

## How to Run

1. Clone the repository
2. Open the project in your preferred Java IDE
3. Compile and run `VUPay.java`
4. Enter a valid Module Code when prompted
